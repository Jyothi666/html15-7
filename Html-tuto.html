Simple HTML Web page

1.HTML stands for Hyper Text Markup Language.

2.Hypertext is text which contains links to other texts.

3.Markup language means set of instructions to display in the browser.

4.HTML create the structure of a Web page,which is static.

5.HTML tree has mainly two branches.

-->Head

-->Body

Tags

The content which is enclosed with < >

1.Opening Tag(< >)

2.Closing Tag(< / >)

Types of Tags

1.Meta Tags

2.Formatting Tags

3.Semantic Tags

4.Multimedia Tags

Element

HTML element is defined by a start tag, some content, and an end tag.

< tagname > Content < /tagname >

1.HTML contains 6 heading tags such as h1,h2,h3,h4,h5,h6.

2.'h1' has high font size whereas 'h6' has small size

Example:

< h1 >Hello< /h1 >

< h2 >Hello< /h2 >

< h3 >Hello< /h3 >

< h4 >Hello< /h4 >

< h5 >Hello< /h5 >

< h6 >Hello< /h6 >

3.For paragraph tag we use < p >< /p >
